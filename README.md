# SpaceX Landing Prediction Project

## Overview

Training project developed as part of the IBM Data Science Professional Certificate.

Objective: predict whether the first stage of a SpaceX Falcon 9 launch will land successfully using historical launch data.

## Business Problem

Predict landing success to estimate launch costs and support decision-making.

## Project Workflow

1. Data Collection (API + Web Scraping)
2. Data Cleaning & Wrangling
3. Exploratory Data Analysis
4. Data Visualization
5. Interactive Dashboard
6. Machine Learning Models
7. Final Conclusions

## Technologies Used

Python, Pandas, NumPy, SQL, Matplotlib, Seaborn, Folium, Plotly Dash, Scikit-learn
