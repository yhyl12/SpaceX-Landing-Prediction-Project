# SpaceX Landing Prediction Project

## Project Overview

This project predicts whether the first stage of a SpaceX Falcon 9 rocket will land successfully using historical launch data.

The project was developed as part of the IBM Data Science Professional Certificate and covers the full Data Science workflow.

## Business Objective

Reusable first-stage landings significantly reduce launch costs. Predicting landing success helps estimate mission efficiency and costs.

## Workflow

1. Data Collection (API + Web Scraping)
2. Data Cleaning & Wrangling
3. Exploratory Data Analysis
4. Data Visualization
5. Interactive Maps and Dashboard
6. Machine Learning Models
7. Final Conclusions

## Tools & Technologies

Python, Pandas, NumPy, SQL, Matplotlib, Seaborn, Folium, Plotly Dash, Scikit-learn

## Key Results

* Built classification models to predict landing success
* Developed interactive dashboards
* Performed full end-to-end analysis
